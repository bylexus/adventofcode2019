# adventofcode2019
My adventofcode.com solutions for 2019

This year I challenge myself by using C++ as solving language - whenever possible :-)


## Compile

I'm using cmake for the build infrastructure. So run:

```bash
$ cmake -S . -B build
$ cd build
$ make
```
